# 🚀 快速测试指南

## 启动项目

```bash
# 1. 确保开发服务器正在运行
npm run dev

# 2. 在浏览器打开以下页面测试
```

---

## 📱 页面测试清单

### ✅ **首页** - http://localhost:3000
- 应该看到 8 个导航分类
- 应该看到 40+ 个网站卡片
- 点击侧边栏分类应该平滑滚动
- 滚动页面时侧边栏应该自动高亮
- 点击网站卡片应该能打开网站

### ✅ **文章列表** - http://localhost:3000/articles  
- 应该看到 8 篇文章
- 点击左侧分类应该筛选文章
- 应该看到热门标签
- 应该看到推荐文章

### ✅ **文章详情** - http://localhost:3000/posts/react-19-new-features
- 应该看到文章内容
- 应该看到相关文章推荐
- 刷新后浏览量应该+1

---

## 🔍 API 测试

在浏览器直接访问：

- http://localhost:3000/api/categories - 获取导航分类
- http://localhost:3000/api/websites - 获取网站列表  
- http://localhost:3000/api/articles - 获取文章列表
- http://localhost:3000/api/tags - 获取标签列表

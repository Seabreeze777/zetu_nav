## 🚀 启动项目前的准备工作

### ⚠️ 重要：数据库更新

由于添加了**收藏功能**，数据库结构有变化，**必须**先同步数据库：

```bash
npx prisma db push
```

这个命令会创建以下新表：
- `website_favorites` - 网站收藏表
- `article_favorites` - 文章收藏表

### 📦 确认依赖已安装

如果还没有安装依赖，请运行：

```bash
npm install
```

### 🗄️ 确认数据库已配置

检查 `.env` 文件中的数据库连接信息：

```env
DATABASE_URL="mysql://zetu-nav:0dae0b6a05@81.70.76.184:3306/zetu-nav"
```

## 🎨 技术栈总览

### 前端
- **框架**：Next.js 14 (App Router)
- **样式**：Tailwind CSS
- **UI组件**：自定义组件
- **Markdown**：react-markdown + 插件

### 后端
- **ORM**：Prisma
- **数据库**：MySQL
- **认证**：JWT + Cookie
- **密码加密**：bcryptjs

### 存储
- **图片存储**：腾讯云 COS（需配置）
- **数据库**：MySQL（腾讯云）

## 🐛 已知问题 & 注意事项

### 图片相关
- 目前图片URL需要手动输入
- 建议使用腾讯云COS或其他CDN
- `ImageWithFallback` 组件已处理图片加载失败

### 权限相关
- 普通用户无法访问后台管理
- 管理员可以访问所有功能
- 角色修改需要在用户管理中操作

### 数据相关
- 删除分类前需要先删除关联的内容
- 删除标签前需要先解除文章关联
- 用户不能删除自己的账号

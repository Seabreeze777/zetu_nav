# 🎨 设计风格预览

这个目录包含 5 种不同设计风格的示例页面，用于选择最终的 UI 设计方案。

## 📂 文件说明

- `style-a-极简现代.html` - 方案 A：现代极简风格
- `style-b-卡片设计.html` - 方案 B：卡片式设计
- `style-c-玻璃态.html` - 方案 C：玻璃态/毛玻璃风格
- `style-d-深色主题.html` - 方案 D：深色主题
- `style-e-活力彩色.html` - 方案 E：活力彩色风格

## 🚀 如何查看

**方法 1：直接双击 HTML 文件**
- 找到对应的 `.html` 文件
- 双击用浏览器打开

**方法 2：在文件管理器中右键**
- 右键 HTML 文件
- 选择"打开方式" → 选择你的浏览器

**方法 3：拖拽到浏览器**
- 打开浏览器
- 把 HTML 文件拖进去

## ⚠️ 注意

这些文件是**纯展示用途**，不依赖项目的任何代码，可以随时删除整个 `design-preview` 目录，不会影响项目运行！
